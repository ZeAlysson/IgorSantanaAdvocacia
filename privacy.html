Política de Privacidade

Esta Política de Privacidade descreve como Igor Santana Advocacia (doravante "nós", "nosso") coleta, utiliza, compartilha e protege os dados pessoais dos visitantes e clientes do site. Ao utilizar este site, você concorda com as práticas descritas nesta política.

1. Dados Pessoais Coletados

Coletamos e armazenamos determinados dados pessoais com a finalidade de prestar serviços e melhorar a experiência do usuário. Os tipos de dados pessoais podem incluir, mas não se limitam a:

  • Nome
  • Endereço de e-mail
  • Número de telefone
  • Informações de contato enviadas pelo formulário

2. Uso dos Dados Pessoais

Os dados pessoais coletados serão utilizados para os seguintes fins:

  • Fornecer informações sobre nossos serviços jurídicos e atualizações relevantes.
  • Personalizar a experiência do usuário no site.
  • Melhorar nossos serviços e o próprio site com base em análises de uso.

3. Compartilhamento de Dados Pessoais

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros para fins comerciais indiscriminados. Contudo, podemos compartilhar dados pessoais com as seguintes entidades quando necessário:

  • Parceiros de publicidade, como Facebook Ads e Google Ads, para veiculação de anúncios direcionados (quando aplicável).
  • Provedores de serviços que auxiliam na operação do site (por exemplo, serviços de hospedagem e ferramentas de análise).
  • Autoridades legais, quando exigido por lei ou para proteger nossos direitos.

4. Segurança dos Dados

Adotamos medidas técnicas e organizacionais razoáveis para proteger os dados pessoais contra acesso não autorizado, alteração, divulgação ou destruição. Entretanto, nenhum sistema é completamente invulnerável; em caso de incidentes, comunicaremos as autoridades competentes conforme a legislação aplicável.

5. Retenção de Dados

Reteremos seus dados pessoais apenas pelo tempo necessário para cumprir as finalidades descritas nesta política, a menos que a lei exija um período de retenção maior.

6. Direitos dos Titulares

Você tem o direito de acessar, corrigir, solicitar a portabilidade ou a exclusão de seus dados pessoais, bem como de se opor ao tratamento ou solicitar a limitação do tratamento quando aplicável. Para exercer esses direitos, entre em contato conosco através das informações abaixo.
